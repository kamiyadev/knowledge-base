# Pythonにおけるイベントループとは

Pythonにおける**イベントループ（event loop）**とは、  
**発生したイベント（処理要求）を順番に監視し、適切な処理を実行し続ける仕組み**のことです。

特に **非同期処理（async / await）** と深く関係しています。

---

## イベントループを一言で
👉 **「処理待ちの仕事（イベント）を管理・実行する司令塔」**

---

## なぜイベントループが必要？
通常のPython処理は **上から順に同期的に実行** されます。

download()
process()
