Graph API 呼び出しを実施する

curl の実行結果を確認する
  curl の終了コードが 0 以外の場合
    ネットワークエラーと判定
    エラーログを出力し処理を終了する

HTTP ステータスコードを確認する
  200系の場合
    ユーザー一覧情報を取得し正常終了する

  200系以外の場合
    Entra ID からの論理エラーと判定する
    レスポンス JSON の error.code / error.message をログ出力する

    ステータスコードに応じて以下の処理を行う
      401：認証エラーとして処理終了
      403：権限エラーとして処理終了
      429：一定時間待機後リトライ
      その他：想定外エラーとして処理終了
